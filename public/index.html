このサイトはOSSのSNSエンジン Flockbird を使用したデモサイトです。

  • Flockbird はOSSのSNSエンジンです
  • FacebookライクなSNSを構築できます
  • FuelPHP1.7.X を Framework に使用しています
  • Licenseは MIT ライセンスです
  • Github

ログインアカウント

  • 投稿にはログインが必要です
  • 下記のURL・アカウントでログインできます
    • URL : http://demo.flockbird.uzuralife.com/auth/login
    • email: sample@example.com
    • pass : password
  • 自由に新規登録することも可能です

管理画面

  • 管理画面は下記のURL・アカウントでログインできます
    • URL : http://demo.flockbird.uzuralife.com/admin
    • email: admin
    • pass : password

注意事項

  • 入力したデータは一定時間で削除します
